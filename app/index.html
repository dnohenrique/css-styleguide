Formulário

Formulários devem começar com a classe .form-group

        
  <div class="form-group"> 
    <div class="form-field">
      <label class="field-description">Normal input text
      <input type="text" class="input-field">
    </div>
  </div>
      
        
  <div class="form-group form-rh"> 
    <input type="text" class="input-field">
  </div>
      
        
  <div class="form-field">
    <label class="field-description">Select</label>
    <select class="input-field">
      <option value="">-- Nenhum --</option>
      <option value="x">Presidente / Proprietário</option>
      <option value="y">Diretor</option>
    </select>
  </div>
Error Message. 
        
  <input type="text" placeholder="Input Error" class="input-field input-error">
  <div class="error-message">
    Error Message.
  </div>
      
        
  <!-- Para radio button de RH adicionar acima do elemento a classe .form-rh -->
  <label class="fc-input-radio">
    <input type="radio" name="umradio">
    <span class="fc-checkradio"></span>
    Sim
  </label>
      
          
  <!-- Para radio button de RH adicionar acima do elemento a classe .form-rh -->
  <div class="form-field">
    <div class="checkbox_field">
      <label>Um checkbox</label>
      <input type="checkbox" checked="true">
      <span class="checkmark"></span>
    </div>            
  </div>

Progress bar

Use .progess-color-{uma cor} para definir a cor da barra de progresso.

As classes de cores disponíveis são:

  • .progress-color-passeio-noturno
  • .progress-color-horizonte
  • .progress-color-mergulho
  • .progress-color-ceu-de-pipa
  • .progress-color-por-do-sol
  • .progress-color-black
